# 🏥 Hospital Readmission Prediction

## 📌 Problem Statement

Predict whether a patient will be readmitted within 30 days using healthcare data.

---

## 📊 Dataset

* Diabetes dataset (US hospitals)
* ~100,000 patient records

---

## ⚙️ Approach

* Data Cleaning & Preprocessing
* Handling Missing Values
* Feature Engineering
* Handling Class Imbalance
* Logistic Regression Model

---

## 📈 Key Insights

* More medications → higher readmission risk
* Longer hospital stay → higher risk
* Previous inpatient visits strongly impact readmission
* Discharge disposition is a key factor

---

## 📊 Results

* Initial model failed due to imbalance
* Improved model achieved ~41% recall for high-risk patients

---

## 🧰 Tools Used

* Python
* Pandas, NumPy
* Scikit-learn
* Matplotlib, Seaborn
